# Go Lab

A collection of Go mini projects, coding exercises, and experiments built while learning Go.

The goal of this repository is to document my journey from Go fundamentals to production-ready backend development through hands-on projects.

## Objectives

- Learn Go by building real applications
- Master Go's core language features
- Explore idiomatic Go practices
- Build reusable utilities and tools
- Prepare for large-scale backend development


## Project Structure

```
go-lab/
│
├── 01-hello-world/
├── 02-cli-calculator/
├── 03-number-guess/
├── 04-password-generator/
├── 05-file-organizer/
├── 06-json-parser/
├── 07-url-shortener/
├── ...
└── README.md
```

Each project is self-contained and focuses on one or more Go concepts learned throughout the journey.

## Tech Stack

- Go
- Standard Library
- Git

Additional technologies such as PostgreSQL, Docker, Redis, and AWS will be explored in separate production projects.

## Goal

Build a strong foundation in Go through consistent practice and project-based learning.
